## Overview

### The challenge 📝

### Built with 🛠

-Nextjs 
-TypeScript
- Functional Programming
- Axios
- Eslint as the linter
- Prettier as the code formatter


### Development

First to run 
```
npm i
```
After all dependencies installed, run
```
npm run dev
```

### Production

```
npm run build
```
